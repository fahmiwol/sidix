## Konteks

Apa saja layer/file yang membentuk sistem NPC Mighantect? Jangan rebuild dari nol!

## Pengetahuan

Sistem NPC Mighantect sudah dibangun berlapis oleh Cursor. Jangan rebuild — integrasikan: (1) HR Ledger: config/agent-ledger.json + server/agent-ledger.js — buku besar NPC dengan bio, gender, npcPersona (archetype/catchphrase/taboo/dream), voicePersona, uiFun (cardTagline/slackStatus/coffeeOrder), manualSkills, manualModules, portraitImagePrompt. (2) Personality: server/agent-personality.js — traits, speechStyle, greetings, catchphrases, moodResponses per agent. (3) Capabilities: server/agent-capabilities.js — skills/tools per agent + AGENT_MEDIA_TOOLS map. (4) Memory: server/memory-store.js — short/long-term memory storage per agent. (5) Settings: server/agent-settings.js + server/.data/agent-settings.json — provider+model routing runtime. (6) Emotion: server/agent-autonomy.js — 7 emotion states, decay, Socket.io broadcast /api/emotion/*. (7) Learning: server/agent-learning.js + config/agent-learning-config.json — self-learn loop, 25 agents dengan search queries. (8) World def: config/world.json agentDefs — visual, spawn position, building assignment. Sprint 9A menambah npc-module.js + npc-registry.js + config/npc-profiles/ yang PERLU diintegrasikan sebagai facade/wrapper layer di atas semua ini, bukan menggantikan.
