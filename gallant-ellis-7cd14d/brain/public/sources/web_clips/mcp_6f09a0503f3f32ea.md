## Pengetahuan

Module: brain_qa/memory.py. Fungsi utama: retrieve_relevant_cards(question, top_n=4, score_threshold=0.08) → list[MemoryCard]. Cara kerja: (1) Load brain/datasets/memory_cards.jsonl, (2) Score tiap card berdasarkan weighted token overlap (tag match bobot 3x lebih tinggi dari content overlap), (3) Kartu wajib selalu disertakan: principle-001, preference-001, ihos-002, (4) format_memory_context(cards) → string blok "Konteks Memori". Integrasi di query.py: dipanggil di answer_query_and_citations() SEBELUM persona block — memory context jadi bagian pertama output. Setelah ini M3 semua checklist selesai: ingest ✅, BM25 index ✅, retrieve+cite ✅, QA eval ✅, memory injection ✅.
