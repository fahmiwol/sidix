## Konteks

Bagaimana 7 metode mengajar Rasulullah SAW diterjemahkan menjadi arsitektur pembelajaran SIDIX?

## Pengetahuan

7 METODE RASULULLAH SAW → PADANAN ARSITEKTUR SIDIX:

1. KETELADANAN (Uswah Hasanah) → SPIN Imitation Learning
   - Human response = "teladan" (winner), SIDIX lama = loser
   - Constitutional check sebelum setiap output
   - Konsistensi: prinsip sama di semua konteks
   - Kode: UswahHasanahTrainer.build_spin_pair(human_response, sidix_response)

2. BERTAHAP (Tadarruj) → Curriculum Learning
   - 4 level: L1_foundational → L2_intermediate → L3_advanced → L4_synthesis
   - Setiap training pair diberi difficulty_level
   - Advance ke level berikutnya hanya jika score >= 0.75 di level sebelumnya
   - Adaptive response: deteksi level penanya → sesuaikan kompleksitas jawaban

3. NASIHAT (Mau'izhah) → Constitutional AI
   - SIDIX_CONSTITUTION: 12 prinsip tertulis sebagai reward function
   - Prinsip: jujur meski pahit, akui ketidaktahuan, manfaat nyata, IHOS sanad
   - Mau'izhah hasanah: kritik selalu disertai solusi konstruktif

4. DIALOG & TANYA JAWAB → Q&A Training Pairs + Multi-turn RLHF
   - Upgrade dari single-turn ke multi-turn dialogue templates
   - Socratic dialog: pertanyaan → jawaban L1 → pertanyaan lanjutan → jawaban L2
   - Safe environment: tidak ada pertanyaan yang "bodoh"
   - Log semua pertanyaan sebagai gap data → curriculum

5. PERUMPAMAAN (Amtsal) → Analogy-based RAG
   - AmtsalLibrary: database (konsep abstrak → perumpamaan konkret)
   - RAG pipeline: retrieve context → cek AmtsalLibrary → inject analogi → generate
   - Auto-extract analogies dari corpus via LLM
   - Ranking analogi berdasarkan latar belakang user

6. DEMONSTRASI/PRAKTIK → Tool Use + Validated Skill Library
   - Show don't tell: selalu sertakan kode runnable, bukan hanya deskripsi
   - Voyager skill library = "kitab demonstrasi" SIDIX
   - Hanya kode yang sudah divalidasi (tested=True) yang dijadikan teladan
   - Unvalidated code: diberi disclaimer

7. TARGHIB-TARHIB (Reward & Punishment) → Self-Reward Scoring
   - 5 kriteria: accuracy(30%), helpfulness(25%), clarity(20%), completeness(15%), constitutional(10%)
   - Verdict: targhib_high (>0.85) → SPIN winner | tarhib (<0.50) → SPIN loser
   - Adil: tidak menghukum jawaban sederhana yang tepat sasaran

KONSEP TAMBAHAN:
- Al-Zarnuji Knowledge Authority Hierarchy: Tier 0 (Constitution) → Tier 1 (peer-reviewed) → Tier 2 (expert) → Tier 3 (community) → Tier 4 (unverified)
- Ibn Khaldun: practical_skill > conceptual_understanding > factual_knowledge (soft skill prioritas)
- Ibn Khaldun: guru tidak berhenti belajar = justifikasi self-evolving loop SIDIX

FULL PIPELINE dengan Prophetic Pedagogy Layer:
Input → [Ibn Khaldun: deteksi level] → [Tadarruj: pilih difficulty] → [Amtsal: cari analogi] → [Dialog: klarifikasi jika ambigu] → [Demonstrasi: contoh konkret] → [Keteladanan: constitutional check] → [Nasihat: tambah dimensi manfaat] → Output → [Targhib-Tarhib: scoring] → [Flywheel: best pairs ke SPIN]
