**Sumber:** claude-chat  
**Tanggal:** 2026-04-17  
**Tags:** php, curl, integration, crm, sidix  

## Konteks

Bagaimana cara memanggil SIDIX dari aplikasi PHP tanpa framework?

## Pengetahuan

Buat helper core/sidix.php dengan fungsi: sidix_capture($topic, $content) untuk rekam pengetahuan, sidix_query($question) untuk tanya. Implementasi pakai curl dengan timeout 5 detik (CURLOPT_TIMEOUT=5, CURLOPT_CONNECTTIMEOUT=3) agar tidak memblok proses utama. Endpoint: POST /corpus/capture untuk simpan, POST /agent/chat untuk tanya. Panggil sidix_capture() setelah operasi penting (lead baru, FU selesai, status berubah) sebagai fire-and-forget. Timeout pendek memastikan kalau SIDIX offline pun aplikasi tetap jalan normal.
