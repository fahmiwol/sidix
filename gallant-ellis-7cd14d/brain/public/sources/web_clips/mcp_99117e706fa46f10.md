## Konteks

Bagaimana cara implementasi inline edit, date picker, dan day dropdown di tabel PHP tanpa framework?

## Pengetahuan

Pattern inline UX di PHP tabel tanpa framework: 1) Day Dropdown: select dengan color per option via CSS, onchange kirim AJAX ke save_fu_day.php, hitung next weekday dengan strtotime('next monday'). 2) Date Picker: hidden input type=date, tampilkan div yang clickable, onclick panggil inp.showPicker() atau inp.click(), onchange AJAX ke save endpoint. 3) Inline Edit Textarea: dua div (view-mode/edit-mode), toggle display none/block saat klik, textarea dengan tombol Simpan+Batal, AJAX fetch POST ke endpoint PHP. Endpoint PHP pakai ALTER TABLE IF NOT EXISTS untuk tambah kolom baru secara idempotent. Penting: stopPropagation() di semua interaksi agar tidak trigger row click handler yang membuka panel lain.
