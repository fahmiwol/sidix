# NPC avatar SVG sprite generator

## Konteks

Bagaimana generate avatar NPC sebagai inline SVG tanpa server/image file?

## Pengetahuan

Pattern `_npcSvgAvatar(id, name, size)` di admin panel: (1) Deterministic color dari charCode sum modulo palette array; (2) SVG element: background rounded rect (bgColor), hair ellipse (dark color, top of head), head circle (skin tone #f5c59d), body rect (shirtColor = agent color), initials text overlay; (3) Return `data:image/svg+xml;charset=utf-8,` + encodeURIComponent(svg); (4) Used as `<img src="...svg data url...">` — works in all browsers; (5) Helper `_npcSvgAvatarImg(id,name,size)` returns full `<img>` tag. Key formula: headCY = h*0.30, headR = w*0.22, bodyY = headCY+headR+1, bodyW = w*0.46, bodyH = h*0.32. Base64 encode not needed — percent-encode works for inline SVG.
