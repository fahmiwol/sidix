## Pengetahuan

SIDIX bukan wrapper OpenAI/Anthropic. SIDIX adalah LLM sendiri yang dibangun dari nol dengan prinsip lokal, privat, tanpa vendor lock-in.

Tujuan utama:
1. SIDIX = model generatif yang dilatih dari corpus kita sendiri (research notes, framework, knowledge base)
2. Fine-tune via Kaggle GPU (QLoRA) → hasilkan LoRA adapter → deploy ke VPS dengan Ollama
3. Framework IHOS/Sanad/Maqasid bukan hiasan — harus jadi decision logic nyata di brain_qa
4. Research notes di brain/public/research_notes/ = SIDIX mengajari dirinya sendiri

Stack saat ini:
- Ollama (qwen2.5:7b) di VPS port 11434 → inference lokal
- brain_qa FastAPI port 8765 → RAG (BM25 + Ollama generate)
- Telegram bot @sidixlab_bot → interface pengguna
- Kaggle T4 x2 → GPU fine-tuning QLoRA

Milestone berikutnya:
- Jalankan Kaggle notebook yang sudah difix → dapatkan sidix-lora-adapter.zip
- Upload adapter ke VPS → /tmp/sidix/apps/brain_qa/models/sidix-lora-adapter/
- Integrasikan adapter ke Ollama sebagai model custom
- Terapkan IHOS/Maqasid sebagai logika reasoning di agent_react.py
