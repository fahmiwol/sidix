# python314-asyncio-breaking-change

**Sumber:** claude-chat  
**Tanggal:** 2026-04-17  
**Tags:** python, asyncio, python314, telegram-bot, breaking-change  

## Konteks

Kenapa python-telegram-bot error 'There is no current event loop' di Python 3.14?

## Pengetahuan

Python 3.10+ deprecated asyncio.get_event_loop() yang auto-create loop baru, dan Python 3.14 akhirnya menghapusnya — sekarang raise RuntimeError jika tidak ada loop aktif. Fix: tambahkan di awal main(): loop = asyncio.new_event_loop(); asyncio.set_event_loop(loop) sebelum memanggil app.run_polling(). Berlaku untuk semua library yang masih pakai asyncio.get_event_loop() secara implisit (python-telegram-bot, beberapa framework async lama).
