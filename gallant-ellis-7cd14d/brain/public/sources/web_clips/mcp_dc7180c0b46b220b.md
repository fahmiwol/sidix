## Konteks

Bagaimana cara evaluasi kualitas retrieval BM25 di brain_qa secara otomatis?

## Pengetahuan

Script eval_qa.py di apps/brain_qa/eval_qa.py. Cara kerja: (1) Load qa_pairs.jsonl, (2) Untuk setiap pair: tokenize question → BM25 retrieve top-k chunks, (3) Hitung token overlap antara meaningful tokens dari retrieved_text vs ideal_answer, (4) Threshold default 12% overlap = PASS. Metric: hit@k rate. Output: PASS/MISS per pair + hit_rate summary + simpan JSON ke .data/qa_eval_result.json. Stopwords ID+EN difilter. Meaningful tokens = token bukan stopword dan panjang > 2 char. Jalankan: python eval_qa.py --k 5 --threshold 0.12 --verbose. M3 dianggap bagus jika hit_rate >= 80%.
