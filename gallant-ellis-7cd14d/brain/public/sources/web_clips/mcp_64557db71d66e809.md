# nextjs-server-actions-pattern

**Sumber:** claude-desktop  
**Tanggal:** 2026-04-16  
**Tags:**   

## Konteks

Bagaimana cara connect Next.js App Router client pages ke backend API yang butuh auth, tanpa expose credentials ke browser?

## Pengetahuan

Pakai Next.js Server Actions ('use server'). Flow: 1) Server Action berjalan di server — bisa pakai env vars sensitif (INTERNAL_API_KEY). 2) Client component import server action, panggil seperti async function biasa. 3) Di dalam server action: ambil JWT dulu via internal key (POST /auth/token dengan x-internal-key), lalu pakai JWT untuk call API yang butuh auth. 4) Credentials tidak pernah ke browser. Pattern di Dompet-tiranyx: getWalletBalance(WALLET_ID) → getBankJwt(userId) → bankTransfer/bankTopup/bankWithdraw. Untuk halaman yang butuh data server-side (tarik/riwayat): jadikan Server Component wrapper yang fetch data → pass ke Client Component sebagai props.
