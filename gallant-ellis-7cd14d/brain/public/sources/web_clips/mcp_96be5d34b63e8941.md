**Tags:**   

## Konteks

Bagaimana cara setup SIDIX MCP Server sebagai plugin Claude Desktop di Windows agar aktif di semua sesi?

## Pengetahuan

1. Buat MCP server dengan @modelcontextprotocol/sdk (stdio transport) di sidix-mcp/server.js. 2. Install deps: cd sidix-mcp && npm install. 3. Tutup Claude Desktop sepenuhnya. 4. Edit %APPDATA%\Claude\claude_desktop_config.json — tambahkan mcpServers block di root level (bukan dalam preferences). 5. Buka Claude Desktop — MCP auto-connect. PENTING: jangan edit config saat Desktop masih berjalan. Scope valid: local, user, project (bukan global). Storage: %USERPROFILE%\.claude\sidix-knowledge\
