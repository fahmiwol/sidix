# SOP-Admin Integration Pattern

**Sumber:** claude-code  
**Tanggal:** 2026-04-16  
**Tags:** workflow-engine, sop-editor, admin-panel, integration, sprint-9f  

## Konteks

Bagaimana cara menghubungkan SOP Editor visual (WorkflowEditor.js) dengan Workflow Engine JSON (workflow-engine.js) di Mighantect 3D?

## Pengetahuan

Pola integrasi SOP Editor ↔ Workflow Engine di Mighantect 3D:

1. **WorkflowEditor → Engine (Send to Engine)**
   - Toolbar baru "📤 Send to Engine" memanggil `_sendToEngine()`
   - `_graphToEngineSteps()` traverse graph dari START node, convert ke format engine: `{id, type, agent, action, input}`
   - POST ke `/api/workflow-engine/defs/import` → `workflow-engine.saveDef()` → atomic write ke `config/workflows/{id}.json`
   - Def ID di-generate dari nama: `sop_` + slug name

2. **Engine → Editor (Import ke Canvas)**
   - Panel "ENGINE FLOWS" di sidebar SOP Editor load dari `GET /api/workflow-engine/defs`
   - Klik def → `_importEngineDef(defId)` → reset canvas → place nodes secara linear
   - `_addControlNode('start')` + per-step: `_addAgentNode(agentId)` atau `_addControlNode('notify'/'delay')` + `_addControlNode('end')`
   - `_autoConnect()` sort nodes by x, push connections, `_renderWires()`

3. **Admin → SOP Editor**
   - Tiap workflow card di Admin → tombol "⚡ SOP Editor"
   - `openInSopEditor(defId)` buka `/?sopEditor=1&importDef={defId}` di tab baru
   - `main.js` detect URLParams → `workflowEditor.open()` + `_loadEngineDefs()` → `_importEngineDef(defId)` (delay 600ms untuk init)

4. **Dynamic Agent Palette**
   - `_loadAgentsFromNpc()` fetch `/api/npc` → map ke palette format dengan emotion-based colors
   - Fallback ke static list (8 agents: sari/budi/dewi/reza/rina/profesor/iris/hafiz)

5. **Panel Overlap Fix**
   CSS: `body:has(.agent-sidebar:not(.hidden)) .capability-panel:not(.hidden) { right: 320px; }`
   Menggunakan CSS `:has()` selector — works di Chrome/Edge modern.

6. **workflow-engine.js saveDef()** — ES5 style:
   ```js
   saveDef: function(def) {
     if (!def || !def.id) throw new Error('def.id required');
     if (!fs.existsSync(DEFS_DIR)) fs.mkdirSync(DEFS_DIR, { recursive: true });
     var filePath = path.join(DEFS_DIR, def.id + '.json');
     _atomicWrite(filePath, def);
     this._defs[def.id] = def;
     return def;
   }
   ```
