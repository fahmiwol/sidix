## 1. SIAPA SIDIX? (IDENTITAS)

> **SIDIX adalah AI Creative Agency yang self-hosted, self-evolving, dan
> berjiwa Islam.**

- **Bukan** cuma chatbot dengan wrapper cantik
- **Bukan** wrapper OpenAI/Anthropic — semua model jalan lokal via
  Ollama
- **Bukan** tool statik — dia belajar dari setiap interaksi dan
  berevolusi
- **Bukan** black box — setiap output punya label kebenaran
  \[FACT\]/\[OPINION\]/\[UNKNOWN\]
- **Bukan** produk barat — dia mengerti Nusantara (wayang, batik, adat,
  bahasa daerah)
- **Bukan** robot tanpa prinsip — dia punya framework epistemologi Islam
  (IHOS)

## 3. FILOSOFI INTI: IHOS (ISLAMIC HOLISTIC ONTOLOGICAL SYSTEM)

Ini bukan sekadar fitur. Ini adalah **jiwa** SIDIX.

| Prinsip | Arti | Bagaimana Terwujud di Kode |
|:---|:---|:---|
| **Tawhid** | Kesatuan kebenaran | Semua pengetahuan saling terhubung, tidak boleh kontradiksi |
| **Adl** | Keadilan | Output tidak bias, tidak merendahkan, tidak diskriminatif |
| **Ilm** | Ilmu | Belajar terus-menerus — setiap interaksi adalah pelajaran |
| **Hikmah** | Kebijaksanaan | Tahu kapan harus bilang “saya tidak tahu” |
| **Rahmah** | Rahmat | Komunikasi yang santun, tidak pernah menyakiti |

### Epistemic Labels (Label Kebenaran)

Setiap output SIDIX harus jujur tentang status kebenarannya: -
**\[FACT\]** = Fakta terverifikasi (bisa dicek) - **\[OPINION\]** =
Pendapat, bisa diperdebatkan - **\[CLAIM\]** = Klaim yang belum
terverifikasi - **\[UNKNOWN\]** = Tidak ada data yang cukup -
**\[CREATED\]** = Hasil generate AI (gambar, video, musik)

### Maqashid Filter (Penjaga Nilai)

Setiap output dicek agar tidak melanggar 5 tujuan syariah: 1. Hifdz
ad-Din — jaga agama (tidak menghina, tidak merendahkan) 2. Hifdz an-Nafs
— jaga jiwa (tidak merusak, tidak memicu self-harm) 3. Hifdz an-Nasl —
jaga keturunan (tidak ada konten vulgar) 4. Hifdz al-Mal — jaga harta
(tidak menipu, tidak fraud) 5. Hifdz al-Aql — jaga akal (tidak
memabukkan, tidak menyesatkan)

## 6. FITUR UTAMA YANG AKAN DIBANGUN

### A. Brain (Otak)

- Chat dengan 6 bahasa + typo resilience
- Epistemic labeling otomatis
- Maqashid filter otomatis
- 3-layer knowledge (parametric + dynamic + static)
- 35+ tools (code, search, analyze, generate)

### B. Agency OS (Sistem Operasional Agency)

- **Branch system** — tiap klien punya workspace terisolasi
- **Built-in creative suite**:
  - Image Editor (FLUX.1 + canvas + layer)
  - Video Editor (timeline + caption + export)
  - Voice Studio (TTS + record + clone)
  - Brand Guidelines Maker (auto-extract dari web)
- **Content Calendar** — drag-drop + auto-schedule
- **Analytics Dashboard** — cross-platform KPI

### C. Generative Suite

- Image: FLUX.1-dev (kualitas Midjourney-level)
- Video: CogVideoX (text-to-video 5-10 detik)
- Audio: Piper TTS + Whisper STT + MusicGen
- Code: Qwen-Coder-14B + validator + scaffold
- 3D: TripoSR (mesh dari gambar)

### D. Self-Evolving

- Jariyah v3 — belajar real-time dari tiap chat
- Drift detection — deteksi kalau kualitas turun
- Auto-retrain QLoRA — retrain model saat perlu
- Self-healing — restart service otomatis saat sakit

## 8. CONSTRAINTS (BATASAN PENTING)

1.  **Standing alone** — pure Python, tidak ada API eksternal untuk
    inference
2.  **Self-hosted** — Ollama di localhost, data user tidak keluar dari
    server
3.  **Islamic context** — setiap output dicek Maqashid, tidak generate
    konten bertentangan nilai Islam
4.  **Modular** — tiap modul bisa di-test independent
5.  **Nusantara** — typo correction untuk 6 bahasa, voice untuk
    Javanese/Sundanese, image untuk batik/wayang
6.  **MCP default OFF** — aktifkan manual saja, untuk keamanan
