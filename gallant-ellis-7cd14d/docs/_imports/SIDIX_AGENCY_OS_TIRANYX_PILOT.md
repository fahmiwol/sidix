## 1. VISION: SIDIX AS AGENCY BRAIN

> **“SIDIX bukan lagi cuma AI chatbot. SIDIX adalah Agency Operating
> System — otak utama yang menggerakkan seluruh operasional agency. Dari
> brief klien masuk, sampai asset final deliver, semua dalam satu
> platform. Editor, generator, scheduler, analyst — semua built-in.
> Cabang untuk tiap klien. Sidebar untuk tiap tool.”**

### What This Means for Tiranyx

Tiranyx sudah 10+ tahun handle 200+ brands dengan tim manusia. Sekarang
bayangkan:

- **Satu SIDIX instance** = satu creative director + strategist +
  designer + copywriter + developer + analyst
- **Bercabang per klien** = tiap brand punya workspace isolated sendiri
- **Built-in editor** = nggak perlu export ke Photoshop/Canva/Figma
  untuk edit
- **Sidebar tools** = seperti Kimi agent mode, tapi untuk creative tools
- **Trend harian** = SIDIX yang nge-scrape, bukan intern
- **Auto-report** = tiap minggu laporan performansi auto-generate

**Output:** Tiranyx bisa handle 2x lebih banyak klien dengan tim yang
sama. Atau handle klien yang sama dengan tim 50% lebih kecil.

## 4. BRANCHING SYSTEM — MULTI-KLIEN ARCHITECTURE

### 4.1 The Branch Concept

Tiap klien = satu “branch” (cabang) di SIDIX. Isolated, tapi bisa share
intelligence.

`class`` AgencyBranch:`\
`    ``"""`\
`    Represents one client/workspace in the agency.`\
`    Each branch has isolated memory but can access shared intelligence.`\
`    """`\
`    `\
`    ``def`` ``__init__``(``self``, client_id: ``str``, client_name: ``str``):`\
`        ``self``.client_id ``=`` client_id          ``# "kai-2026", "ocid-digital", etc.`\
`        ``self``.client_name ``=`` client_name       ``# "PT Kereta Api Indonesia"`\
`        `\
`        ``# Isolated per client`\
`        ``self``.brand_knowledge_graph ``=`` {}      ``# Brand-specific facts`\
`        ``self``.asset_library ``=`` []              ``# Generated assets for this client`\
`        ``self``.campaign_history ``=`` []             ``# Past campaigns`\
`        ``self``.performance_data ``=`` {}           ``# Analytics data`\
`        ``self``.preferences ``=`` {}                ``# Client preferences (tone, style, etc.)`\
`        ``self``.approval_workflows ``=`` {}         ``# Who approves what`\
`        `\
`        ``# Access to shared (read-only)`\
`        ``self``.shared_trends ``=`` SharedTrendDB   ``# Cross-industry trends`\
`        ``self``.shared_benchmarks ``=`` SharedBench   ``# What works across brands`\
`        ``self``.platform_updates ``=`` SharedPlat   ``# Meta/TikTok/Google updates`\
\
\
`class`` BranchManager:`\
`    ``"""Manages all client branches."""`\
`    `\
`    ACTIVE_BRANCHES ``=`` {`\
`        ``"kai"``: AgencyBranch(``"kai"``, ``"PT Kereta Api Indonesia"``),`\
`        ``"ocid"``: AgencyBranch(``"ocid"``, ``"OCID Digital"``),`\
`        ``"pgn"``: AgencyBranch(``"pgn"``, ``"PGN (Perusahaan Gas Negara)"``),`\
`        ``"gov-subsidi"``: AgencyBranch(``"gov-subsidi"``, ``"Government Subsidy Program"``),`\
`        ``# ... 200+ brands historically, 10-20 active`\
`    }`\
`    `\
`    ``def`` switch_branch(``self``, branch_id: ``str``):`\
`        ``"""Switch active context. All AI responses use this branch's KG."""`\
`        ``self``.current_branch ``=`` ``self``.ACTIVE_BRANCHES[branch_id]`\
`        ``# Update system prompt dengan brand context`\
`        ``# Load brand guidelines`\
`        ``# Switch asset library`\
`        ``pass`\
`    `\
`    ``def`` create_branch(``self``, new_client: ``dict``):`\
`        ``"""Onboarding klien baru."""`\
`        ``# 1. Extract brand info dari brief`\
`        ``# 2. Auto-generate brand KG dari website/social media`\
`        ``# 3. Set default preferences`\
`        ``# 4. Create folder structure`\
`        ``# 5. Schedule discovery call script`\
`        ``pass`
