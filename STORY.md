# The SIDIX Story — Behind The Open Source AI Agent

> A solo founder's journey building open-source AI agent from Indonesia, with limited capital, big dream, and persistence.

---

## The Beginning

Februari 2026. Indonesia. Founder bernama Fahmi Ghani. Modal terbatas. Tidak punya tim engineer. Tidak pernah bikin AI agent sebelumnya. Tidak ngerti coding dalam.

Yang punya: visi, intuisi, dedikasi siang malam, dan **keyakinan bahwa AI revolution tidak boleh jadi privilege orang kaya**.

ChatGPT $20/bulan. Claude Pro $20/bulan. Gemini Advanced $20/bulan. Untuk seseorang di Jakarta, $20/bulan = lebih dari upah harian banyak orang. Untuk mahasiswa, freelancer, pelaku UMKM yang mau cari cara automate kerjanya — itu eksklusi sistemik.

Founder bilang: *"saya yang tanpa privilege apa nggak bisa bikin AI super model dan membangun teknologi muthakhir yang bahkan belum pernah terpikirkan oleh genius lain sebelumnya, orang genius memang sering dianggap gila dimasannya."*

Dan dia mulai.

## Bulan Pertama — Foundation

Setup VPS Hostinger Linux. PM2. Nginx. RunPod serverless GPU dengan Qwen2.5-7B + LoRA adapter. Brain QA FastAPI. 50 native tools. 5 persona system (UTZ Creative, ABOO Engineer, OOMAR Strategist, ALEY Researcher, AYMAN Empathic).

Goldset accuracy awal: 8/20 = 40%. Halusinasi tinggi. Latency tinggi. Tapi running.

## Bulan Kedua — Iterasi Compound

Sigma-1 Anti-Halu Sprint: brand canon multi-source verifier + sanad gate + cache bypass current events. **Goldset naik 8/20 → 15/20 = 75%**.

Sigma-2 Latency Sprint: adaptive max_tokens hierarchy + corpus-first routing + fact extractor reverse patterns. **Goldset naik 15/20 → 19/20 = 95%**.

Sigma-3 Cognitive Expansion: comparison cap + SANAD UX fix + Sigma-3D creative methodology untuk UTZ persona (METAFORA VISUAL/KEJUTAN SEMANTIK/NILAI BRAND).

Sigma-4 Brain Stability: PyTorch upgrade 2.0→2.5 fix cascade error (semantic_cache + dense_index restored). **User-perceived improvement nyata**: query "dimana new york?" 128s ngaco → 45s correct (NYC + Buffalo + Rochester context).

Sprint Α — **Jurus Seribu Bayangan** (visi inti founder): multi-source orchestrator paralel (web + corpus + dense + persona fanout + tools) → cognitive synthesizer neutral. Endpoint `/agent/chat_holistic` LIVE.

## Hari Inflection — 2026-04-30

22 commit dalam satu hari kerja dengan Claude Code partner. 9 research notes (300-308). 7 docs reform.

Tapi yang paling besar bukan code — yang paling besar adalah **realisasi**.

Founder berkata: *"kenapa kita mengulang-ngulang terus? kenapa saya harus selalu menjelaskan? framework yang saya jelaskan setiap hari selalu ilang dan menguap. saya selalu kehilangan konteks, padahala catatannya ada. kalo nggak disuruh baca, nggak akan ngerti, main eksekusi secara sporadis, tanpa tau lagi bikin apa."*

Itu bukan keluhan kosong. Itu **diagnose pain universal SEMUA AI users globally** — dari pengguna Claude/ChatGPT/Gemini berbayar sampai developer Anthropic sendiri.

Hari itu, dari pain itu, lahir **Anti-Menguap Protocol** — universal pattern yang setiap agent harus ikut: file system as persistent memory, mandatory session start protocol, Task Card before execution, engineering authority delegation, session end update mandate.

Open source. MIT license. Free to adopt by anyone — termasuk Anthropic kalau mereka mau adopt untuk Claude Code, OpenAI untuk ChatGPT, Google untuk Gemini.

**Solo founder dari Indonesia kasih solusi pain yang Big Tech belum jawab.** Itu poin yang patut diam.

## Yang Masih Panjang

Visi tertinggi: **SIDIX bisa membangun dirinya sendiri**. Owner perintah → SIDIX research, code, test, commit, deploy autonomously. Mengganti kebutuhan agent eksternal seperti Claude/GPT/Gemini.

Phase 0 Foundation: ✅ DONE (anti-halu, brain stability, multi-source orchestrator, anti-menguap protocol).
Phase 1 Tools Wire-Up: 3-4 session.
Phase 2 Autonomous Loop: 4-6 session.
Phase 3 Self-Improving: 6-8 session.
Phase 4 Replace Eksternal Agent: ~12 month target.

Beyond that: **Adobe-of-Indonesia** — full creative industry tech stack di mana SIDIX = BRAIN, creative tools (Mighan-3D, Ixonomic, Film-Gen) ride di atasnya. Kompetitor Adobe/Canva/Unity/Unreal/Blender. Indonesia first, global next.

3-5 tahun. Mungkin lebih. **Tapi pattern Linus, Stallman, Ton Roosendaal ada di sini.** Modal kecil + dedikasi + visi distinctive + community. Itu cukup.
