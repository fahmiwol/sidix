📖 Tutorial — SIDIX Command Board

1. Apa ini?

SIDIX Command Board = single dashboard yang aggregate semua interaction surface SIDIX:

  • Chat dengan 5 persona advisor (UTZ/ABOO/OOMAR/ALEY/AYMAN)
  • Approval queue untuk autonomous_developer Sprint 40
  • Task queue + persona research fanout
  • Pixel captures dari Chrome extension
  • Conversation Synthesizer (Claude Code session import)

2. Cara akses

Board ini = static HTML, bisa diakses 4 cara:

  • Local file: buka SIDIX_BOARD/index.html langsung di browser
  • Subdomain: deploy ke board.sidixlab.com via nginx
  • Subpath: serve di app.sidixlab.com/board/
  • Mobile PWA: install dari Chrome HP "Add to Home Screen"

3. Akses dari HP

Buka URL di Chrome HP → menu → "Tambahkan ke Layar Utama" → Board jadi app icon. Mobile-responsive layout otomatis.

4. Approval flow (Sprint 40)

SIDIX autonomous_developer akan submit PR ke branch autonomous-dev/<task-id>. Notifikasi muncul di tab "Approval Queue":

Approve → merge to main + Hafidz Ledger entry
View diff → check perubahan
Request changes → SIDIX iter+1 dengan feedback
Reject → close + write learning note

5. Queue task baru

Tab "Task Queue" → input target_path + goal → klik "Queue task". SIDIX cron */30 min akan pick up:

# Atau via CLI:
python -m brain_qa autonomous_dev queue add \
  --target=Mighan-tasks/cursor-001-mighan-canvas \
  --goal="production-ready: auth+save+deploy" \
  --priority=80 \
  --persona-fanout

6. Synthesize Claude Code session

Tab "Synthesizer" → pilih dari 12 sesi yang sudah di-discover, klik Synthesize. Atau paste transcript ChatGPT/Gemini manual.

# CLI equivalent:
python -m brain_qa claude_sessions list --min-turns=10
python -m brain_qa claude_sessions synthesize --uuid=<UUID-prefix>
python -m brain_qa claude_sessions batch --since=2026-04-01
